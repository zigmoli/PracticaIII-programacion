# Gestión de stock de medicamentos — Actividad 3

Repositorio de la Actividad 3 (grupal) de la asignatura **Fundamentos de Programación**. Implementa un sistema en Python para la gestión del stock de medicamentos de la empresa F@rmaTec

## Descripción del proyecto
Este proyecto gestiona el inventario de medicamentos mediante clases en Python y operaciones sobre archivos CSV. Permite cargar datos, buscar medicamentos recursivamente, actualizar cantidades en stock de un medicamento, eliminar medicamentos caducados y guardar los datos actualizados.

El proyecto aplica conceptos de **programación orientada a objetos**, **recursión**, **gestión de archivos** 

## Estructura del repositorio
- `medicamento.py` — Definición de la clase Medicamento.
- `gestor_stock.py` — Definición de la clase GestorStock y lógica principal.
- `actividad3_stock_medicamentos.csv` — Datos iniciales de ejemplo.
- `actividad3_stock_medicamentos_actualizado.csv` — Archivo de salida con datos actualizados.
- `demo_equipo.ipynb` — Notebook con la demostración del funcionamiento.
